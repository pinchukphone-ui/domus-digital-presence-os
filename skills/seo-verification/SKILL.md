# SEO verification

Для каждой опубликованной страницы проверь HTTP 200, один H1, title, description 50–170 символов, абсолютный self-canonical, hreflang self и опубликованный перевод, корректный `html[lang]`, breadcrumbs, внутренние ссылки и sitemap. Draft не должен попадать в production, sitemap или hreflang.
